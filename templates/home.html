Weather Data API

URL Format: http://127.0.0.1:5000/api/v1/station/date

To get data for one station and one date: http://127.0.0.1:5000/api/v1/10/1988-10-25

To get all data for one station: http://127.0.0.1:5000/api/v1/10

To get all data for one station for one year: http://127.0.0.1:5000/api/v1/yearly/10/1980
